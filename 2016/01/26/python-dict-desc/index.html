Python 字典 dict

python字典是序列化数据非常方便,就像最简单的关系型数据库,可以和list互相嵌套
a = {key:value} 建议还是使用双引号来包裹字符串，这样子以后转存为字符串后再次序列号时方便
dict 有几个关键字: “无序的,可变对象,key唯一,可嵌套”

dict func

######clear() 清楚字典数据 like list clear

get() 从字典中拿出指定的key。如果key不存在不会异常只是为None
items() 把字典里面的key和value 作为一个元组填装在一个迭代器中
pop() 类似redis中的pop，从字典中拿出一个key返回key并删除
popitem() 从字典中拿出key和value返回key并删除

区别于pop()方法,
不需要指定参数，随机返回，且key和value放在一个元组中
